# 🌙 AayatVerse

> **A modern Islamic platform for reading, learning, reflecting, and living with the Quran.**

AayatVerse is a modern, mobile-first Islamic platform built with Next.js and React that aims to provide a beautiful, distraction-free experience for reading the Quran, exploring authentic Islamic knowledge, and discovering Quranic guidance for everyday life.

> **⚠️ This project is currently under active development. New features are added regularly.**

---
## 🚀 Live 

https://AayatVerse.com

## ✨ Vision

AayatVerse is more than just another Quran app.

The goal is to build a complete Islamic ecosystem that combines:

- 📖 Beautiful Quran reading experience
- 🌍 Multiple translations (Tarjuma)
- 📚 Authentic Tafsir
- 🤲 Duas & Daily Adhkar
- 🕌 Prayer Times
- 🧭 Qibla Finder
- 📅 Hijri Calendar
- 🔍 Powerful Quran Search
- 💡 AI-powered Quran Guidance
- 🎥 Islamic Short Videos
- ❤️ Personalized Islamic experience

### Ayah For My Situation

Describe your situation and receive Quranic guidance supported by authentic verses and tafsir.

Examples:

- I'm feeling anxious
- I'm struggling financially
- I lost someone
- I feel hopeless
- I need motivation
